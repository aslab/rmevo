\section{Introduction}

The Robot Morphology Evolutioner is a genetic algorithm toolkit oriented to the exploration 
and optimization of hardware and software configurations for robots in uncertain environments.

It uses genetic evolutionary algorithms to evolve the morphologies of a population of robots.  
Morphologies are represented as a tree structure and the evolutionary framework permits
the use of custom evaluation functions to drive robot evolution. 

This project plays a part in the Robominers project, where it attempts to provide 
infrastructure to support the search for optimal mission-oriented morphologies in 
self-reconfiguring robots.
